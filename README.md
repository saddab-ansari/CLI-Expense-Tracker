## 📝 Project Overview
A comprehensive console-based application developed in **C** to help users track their daily spending. This project utilizes file handling to ensure your financial data persists even after the program is closed. It empowers users to seamlessly record expenses, set strict monthly budgets, and generate detailed analytical reports.

## ✨ Key Features
* **Add Expenses:** Log individual records including ID, Date, Amount, Category, and Notes.
* **View History:** Display all recorded expenses in a clean, tabular console format.
* **Budget Management:** Set a monthly spending limit and receive automated warnings if you exceed your budget.
* **Detailed Reports:** Generate total spending calculations filtered by specific months or years.
* **Data Persistence:** Automatically saves all inputs securely to a local text file (`expenses.txt`).

## 🛠️ Technical Details
* **Language:** C (Standard Library)
* **Core Concepts Applied:**
  * **Structures (`struct`):** For organized data grouping and management.
  * **File I/O (`fopen`, `fprintf`, `fscanf`):** For lightweight database management and persistence.
  * **String Handling:** For input formatting and data validation.
  * **Control Structures:** Extensive use of loops and switch-case menus for seamless navigation.
